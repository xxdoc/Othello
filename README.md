一个黑白棋网络对战游戏
======================

2002 年左右开发的一个比较完整的网络对战游戏，当年可以媲美联众棋牌游戏。其中包括客户端，服务器端，人机对战(AI)，帮助文档，安装文件等，目前均已开源。

--------
